Boids Simulation

This is a simulation of flocking behaviour using the boids algorithm. It is written using p5.js.

The boids will be attracted to the mouse while left click is held, and flee from it when right click is held.
Press Z to add a boid, X to add 5 boids or C to add an obstacle.
