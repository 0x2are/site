Welcome!

Hi, I'm Nick, a computer games programming student at Northumbria University in the north of England. Welcome to the home of any projects that I deem to be interesting, useful or awesome enough to show off.

I started teaching myself to program when I started high school, using some online Java tutorials (shoutout to thenewboston). Since then I've worked on many useful things in all sorts of languages. From games and mobile apps to physics simulations and networking tools.
