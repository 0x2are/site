Welcome!

Hi, I'm Nick, a computer games programming student at Northumbria University in the north of England. Welcome to the home of any projects that I deem to be interesting, useful or awesome enough to show off.

I started teaching myself to program when I started high school, using some online Java tutorials (shoutout to thenewboston). Since then I've worked on many useful things in all sorts of languages. From games and mobile apps to physics simulations and networking tools.

Boids Flocking Simulation (Runs in Browser)

A simulation of flocking behaviour using p5.js. The boids can be attracted to or repelled from the mouse and obstacles can be placed that they will try to avoid.

Chip-8 Emulator

This is a project I made for fun over a couple of weekends to learn about emulators. It is written in C++ using SFML.

ASCII Fractal Generator

This is a console based generator of the mandlebrot set that renders using ASCII letters. I plan to add more fractals and paramaters to this in the future.
